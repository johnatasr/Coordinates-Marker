# Join Challenge

Requisitos : Docker instalado

Instruções:

1 - No diretório raiz execute 'docker-compose up --build'
2 - A SPA estará em 'localhost:3000' e a API em 'localhost:8000'
3 - Desfrute e aproveite !

Tecnologias :
    - Django 3.1
    - React
    - Django RestFramework
    - Postgres
    - Docker
